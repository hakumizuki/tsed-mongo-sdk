# tsed-mongo-sdk

- TsED x MongoDB SDK
- 1 min to setup

See [Ts.ED](https://tsed.io) project for more information.

## Build setup

> **Important!** Ts.ED requires Node >= 10, Express >= 4 and TypeScript >= 3.

```terminal
// install dependencies
$ yarn install

// start server
$ yarn start

// build for production
$ yarn build
$ yarn start:prod
```
